# GitHub Repository Setup Guide

## Step-by-Step Instructions

### Step 1: Create GitHub Account (if needed)
1. Go to https://github.com
2. Click "Sign up"
3. Create your account

### Step 2: Create New Repository on GitHub
1. Log in to GitHub
2. Click the **"+"** icon (top right) → **"New repository"**
3. Fill in:
   - **Repository name**: `bollywoodbio` (or your preferred name)
   - **Description**: "BollywoodBio - Movie ticket booking website"
   - **Visibility**: Choose **Private** (recommended) or **Public**
   - **DO NOT** check "Initialize with README" (we already have files)
4. Click **"Create repository"**

### Step 3: Connect Your Local Project to GitHub
After creating the repo, GitHub will show you commands. We'll run them here.

### Step 4: Push Your Code
The commands will be run in the terminal below.

## What We're Doing

1. ✅ Initialize Git (if not done)
2. ✅ Add all files
3. ✅ Create initial commit
4. ✅ Connect to GitHub
5. ✅ Push code to GitHub

## After Setup

Once your code is on GitHub:
1. Go to Hostinger hPanel
2. Node.js → Create App
3. Choose "Deploy from Git"
4. Enter your GitHub repository URL
5. Deploy! ✅
